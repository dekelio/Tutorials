In this experiment, you will be presented with a series of videos. Each video will belong to one of six types: Jump, Swap, Slide, Contain, Huddle, Rotate. Here are example of these types.

  1. Jump - an object changes place by jumping.

  2. Swap - two different objects swap places.

  3. Slide to place - an object changes place by sliding on the floor.

  4. Contain - one object moves towards a different object and contains it.

  5. Huddle - all objects in the scene move to the same spot.

  6. Rotate in place - an object remains in its spot, rotating. Here, the purple cylinder is rotating (ignore the movement of the red cone).
